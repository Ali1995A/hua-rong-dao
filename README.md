# hua-rong-dao

粉色系、适合 5 岁不识字交互的华容道网页（iPad Pro 一代横屏优先），可部署到 Vercel。

## 开发

```bash
npm install
npm run dev
```

## 构建/预览

```bash
npm run build
npm run preview
```

## 推送到 GitHub

```bash
git remote add origin <your-github-repo-url>
git push -u origin main
```

## 部署到 Vercel

- Import Git Repository → 选择本仓库
- Build Command: `npm run build`
- Output Directory: `dist`
